Results
Key findings from the analysis:

Water type is the most common (112 Pokémon)

Flying type is the rarest (4 Pokémon)

Only 65 out of 800 Pokémon are legendary

Generation 1 has the most Pokémon (166)
- Accuracy: 84.5%

- Best feature: HP
